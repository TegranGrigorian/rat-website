What is RAT?

RAT (Rust Archive Tool) is a simple command-line utility to compress and decompress files and folders using the .tar.gz format. It aims to be small, fast, and script-friendly.

Features

  • Compress folders to .tar.gz
  • Extract archives safely
  • Optionally delete inputs after operation
  • Simple CLI usage and automation-friendly
